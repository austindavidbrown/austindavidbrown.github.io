Exercise 3. We will use the Monotone Convergence Theorem. We claim $s_n$ is monotonically increasing. Hence, $s_1 = \sqrt{2} < \sqrt{2 + \sqrt{\sqrt{2}}} = s_2$ and $s_k < s_{k+1}$ for all $1 \le k \le n - 1$ implies $$ s_{n-1} < s_{n} \\\ s_n = \sqrt{2 + \sqrt{s_{n-1}}} < \sqrt{2 + \sqrt{s_{n}}} = s_{n + 1}. $$ By induction, $s_n$ is monotonically increasing. Next, we show $s_n$ is bounded by 2. Hence, $s_1 = \sqrt{2} < 2$, and $s_k < 2$ for all $1 \le k \le n$ implies $$ s_{n} < 2 = s_1 \\\ \sqrt{2 + \sqrt{s_{n}}} = s_{n+1}< \sqrt{2 + \sqrt{2}} < 2. $$ By induction, $s_n$ is bounded by 2, and hence by Monotone Convergence Theorem, $s_n$ converges.
