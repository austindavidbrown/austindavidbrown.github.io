The Properties of $\mathbb{R}$. 1. $\mathbb{R} \subset \mathbb{Q} \subset \mathbb{R}$ 2. $\mathbb{R}$ is an inductive set. 3. $\mathbb{R}$ has the the Archimedean Property. 4. $\mathbb{R}$ is a field, so has field axioms 5. $\mathbb{R}$ is totally ordered, so order has the trichotemy property, transitivity property. 6. $\mathbb{R}$ is complete, so has the Axiom of Completeness property. 7. $\mathbb{R}$ is a complete ordered field. 8. $\mathbb{R}$ is isomorphic to any other complete ordered field. TADA! The real numbers have so much structure it is so beautiful. Definition. Ordered Set. An ordered set $(S, <)$ is a set with the relation $<$ with the trichotemy and transitiviy axioms. It is interesting that $\le$ is defined as $<$ or $=$, so there is nothing more to do with this symbol. Definition. Supremum. $S$ is an ordered set that is bounded above. There exists an $\alpha \in S$ such that $\alpha$ is an upper bound of $S$ and for all $\gamma$, $\gamma$ is an upper bound of $S$ implies $\alpha \le \gamma$. Interesting to note that supremum exists only on ordered sets. I actually thought it needed to be an ordered field for some reason. Definition. Field. A field is a set $S$ such $(S, +), (S, \cdot)$ form 2 Abelian groups and $S$ have the distribution axiom. This is more of an algebraic way of thinking about it, but it is easier for me to organize things personally. Definition. Ordered Field. A field that is an ordered set. From what I have noticed, there are different ways to handle the completeness of $\mathbb{R}$. Rudin uses the construction of $\mathbb{R}$ with Dedekind cuts, and there is also the construction with Cauchy sequences, but I have not seen this one yet. Personally, I like the axiom. The reason for this is we need an complete ordered field type in math in my opinion, so just make it an axiom. However, Rudin uses Dedekind cuts to Construct the Reals. I watched Francis Su's lectures on the topic while skimming through the proof and found his explanation quite great to say the least. Definition. Axiom of Completeness. Every nonempty set of real numbers that is bounded above has a supremum. Theorem 1.19. Completeness of the Real Field. There exists an ordered field with a least upper bound property. Also, as Rudin states at the end of his chapter, any two ordered fields with the least upper bound property are isomorphic. This means we can just work in $\mathbb{R}$ and make an isomorphism to some other complete ordered field. This next property is extremely useful for proving some sequence stuff in Chapter 3. Definition. Archimedes Axiom. Let $S$ be an ordered field. The following are equivalent: 1. For any $x \in S$, there exists an $n \in \mathbb{N}$ such that $n > x$. 2. For any $x \in S$ with x > 0, there exists $n \in \mathbb{N}$ such that $1/n < x$. Rudin states the property a bit differently, and I am not really sure why. Theorem. $\mathbb{R}$ has the Archimedian Property. For all $x, y \in \mathbb{R}$, there exists an $n \in \mathbb{Z}$ and $n > 0$ such that $nx > y$. Definition. Extended Reals. $\mathbb{R}$ and $+\infty$, and $-\infty$. The main idea here is that $+\infty$, and $-\infty$ are symbols that have axioms. I won't copy the axioms from the text.
